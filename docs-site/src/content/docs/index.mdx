## Quick Example

Create your first WebAssembly component in just a few steps:

```python title="BUILD.bazel"
load("@rules_wasm_component//wit:defs.bzl", "wit_library")
load("@rules_wasm_component//rust:defs.bzl", "rust_wasm_component")

wit_library(
    name = "hello_interfaces",
    srcs = ["hello.wit"],
    package_name = "hello:world@1.0.0",
)

rust_wasm_component(
    name = "hello_component",
    srcs = ["src/lib.rs"],
    wit_bindgen = ":hello_interfaces",
)
```
